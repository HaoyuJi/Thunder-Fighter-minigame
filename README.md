# Thunder-Fighter-minigame
Homemade Thunder Fighter mini game, written very rough because it is my debut work

| 目标   | 功能  | 细节     |
| ------ | ---- | -------- |
| 主菜单  | 难度调节   | 3挡（简单、普通、困难） |
|        | 无敌模式  | 开启/关闭   |
|        | 战机左右摆动，开盾动画   | 定时循环    |
|        | 难度数据存储   |     |
| ------ |      |          | 
| 背景   | 背景滚动   | 三层不同速度 |
|        | 流星  | 粒子特效   |
| ------ |      |          |
